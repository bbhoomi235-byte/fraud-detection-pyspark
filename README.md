# Fraud Detection Pipeline using PySpark

## Project Overview
A machine learning pipeline built with PySpark to detect fraudulent bank transactions in real-time.

## Project Details
- **Company:** Hackveda Solutions Pvt Ltd
- **Client:** Deloitte
- **Intern:** Bhoomika T K
- **Project:** Data Risk Mitigation with PySpark

## Problem Statement
Financial institutions face significant losses due to delayed and inaccurate fraud detection. This pipeline solves that using distributed processing and machine learning.

## Technologies Used
- Python
- PySpark 4.0.1
- Random Forest Classifier
- Kaggle Notebooks

## Dataset
- 5000 simulated bank transactions
- 7 features: amount, hour, foreign flag, device, account age
- ~300 fraud cases out of 5000

## Results
- Model Accuracy: 95%
- AUC Score: 0.97
- Fraud Detection Rate: 90%
- Transactions Processed: 5000

## How to Run
1. Open Kaggle Notebook
2. Run all cells in order
3. View results summary at the end
